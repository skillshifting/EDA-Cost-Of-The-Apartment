# Экспресс-EDA и Линейная регрессия: Прогноз стоимости аренды квартир

Проект содержит пример первичного анализа данных (EDA) и построения базовой модели машинного обучения для прохождения технического собеседования.

### Что сделано:
* **Предобработка данных**: Очищены дубликаты, устранены пропуски (заполнение медианой и модой), обработаны аномалии и выбросы в данных.
* **Feature Engineering**: Выполнено One-Hot кодирование категориальных признаков (`pd.get_dummies`) с защитой от мультиколлинеарности (`drop_first=True`).
* **Корреляционный анализ**: Построена матрица корреляции для оценки линейных зависимостей факторов.
* **Моделирование**: Обучена модель `LinearRegression` из библиотеки `scikit-learn`.

### Результаты модели:
* **R² (Коэффициент детерминации)**: ~0.87 (Модель объясняет 87% дисперсии целевой переменной).
* **RMSE**: ~19.46 тыс. USD (Средняя ошибка прогноза стоимости квартиры).

### Стек технологий:
Python, Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn.
